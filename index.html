关于在印北的十来天

发表于 2017-03-23   |   分类于 路上

很多人说去旅行为何不找个伴，一个人多没意思，所以哪些人基本上也就没去过几个地方了。

印度是很多年前想去的国家。

往返机票4k多，如果从昆明往返加尔各答可以省2k左右，这是我后来才知道的。

去印度的话，敲黑板了，一定要带上止泻药和感冒药。如果你不是土豪的话，这里需要升级，如果你不是超级土豪的话，请自备几包乌江榨菜，丧心病狂的可以带上几斤东北大米，自用救济两不误。请不要对印度美食抱有任何幻想，请不要对印度咖喱抱有任何希望。
刚到印度的时候遇到几个小伙伴，因为行程基本相同，跟他们走过几段路，后来想想旅程是自己的，关注点不一样，也就分了，也是合久必分，分久必合。

加尔各答没怎么逗留，其实每个地方都没怎么逗留，一切随缘。印象深的是在恒河，后来出了恒河就再没有哪种感觉了，被虐的感觉。恒河对我而言只是很普通的一条河，上面是尼泊尔，烧尸庙，上上面是冈仁波齐，我也理解什么情况，意思是经过无数
菩提树下的婆娑细语

好了，作为一名专业屌丝，在机场过夜是很平常的一件事，加尔各答的机场比较简陋，过夜还算安全，就是有点冷。

关于西藏

发表于 2017-03-04   |   分类于 路上

对于这个情怀已经聊烂的年代，我想聊一下情怀，关于西藏，拉萨，我们路过的青春、往事。

哪一年是2010年，暴动后的第二个春天，四月，春暖花开。我们是属于第二代进藏，这是按照大姐的说法，她和现在比较火的赵雷是属于第一代，去过西藏的人聊天之中都会不自觉地带点逼气。正如某人说，徒步的看不起骑车的，骑车的看不起搭车的，搭车的看不起坐车的，坐车的看不起坐火车的，坐火车的看不起坐飞机的，坐飞机的看不起所有人一样。
为了装逼，我和老泥手里端着两张开往拉萨的火车票什么都没准备就出发了。其实对于我们来说目标比较单纯，看一下布达拉，装个逼回去吹嘘一下也就够了。至于之后的什么羊湖什么纳木错大昭寺艳遇墙艳遇床我们是什么都不知道。
火车是56个小时，还记得同车是个女同志，去拉萨谋生，她说拉萨人很笨很懒，连土豆皮都不会剥，那边容易做生意哩！哪时的拉萨印象还只是停留在旧照片里面，破破旧旧想想应该就是哪样了。
火车经过唐古拉，高原阳光明媚，老泥的手机放着《坐上火车去拉萨》，哪时觉得相当的带劲，连隔壁箱的旅客也不知不觉啃上几句，坐上火车去拉萨，去看哪神奇的布达拉…

由于火车晚点，入住东措已经很晚，我们的房间是204。90是老房客，90之所以叫90因为他是90，哪个年代也就20不到。由于长得着急并且很老油条，刚开始我还以为90是专业拉皮条，开玩笑啦。还有一姐们叫老孙，后来我们还和老孙一起在长毛的酒吧门口喊过死了都要爱呐。港妹，港妹住在隔壁房间，爱好是睡觉，我们甚至组团去观摩。

藏地的阳光，艳遇墙下的艳遇，喇嘛的转经，八廓街上的讨价还价，累了就去光明茶馆抢地盘，吃着藏面，喝着甜茶，想着藏妹，吹着牛逼。分享淫生，分享艳遇，聊理想也聊淫生，失恋失意失业的三失青年，混在一起的日子并不孤单，萍水相逢的过客，纯纯的友谊。

每天都是聊天喝茶晒太阳，好不无聊，我们也由菜鸟成功转型为老鸟，指导菜鸟成为我们每晚必修课，如何装逼，如何装藏族，如何混进大昭寺，如何才能买到假佛珠假手镯等等都是我们的研究课题。90的皮(饭)条(局)业务也发展到隔壁，隔壁的隔壁，隔壁的隔壁的隔壁。港妹是被发展过来的。港妹来拉萨没有高反，一口气逛完布达拉大昭寺像吃生菜一样，这是她说的。哪天老孙约我们去见识见识尼泊尔餐顺便也约上了港妹，分摊消费，还不是因为穷吗。第一次见港妹，脚踏唐鞋，衣服是混搭，斜挎为人民服务背包，头戴牛仔帽，甚是有型，对于一个素食主义者来说还随身自带筷子，节约环保，我也是服了。后来老泥和港妹去了羊湖，我只好去纳木错，唉，好个重色轻友啊。
tv哥和小凳子也是90发展出来的。tv哥之所以叫tv哥是因为他衣服上面写着cctv，不是ccav。我们都是tv哥的忠实听众，tv哥是徒步拉萨，tv哥光芒万丈，tv哥开讲徒步经历万人空巷，我们都是90度仰望tv哥，tv哥是万人迷，tv哥把90的妹子全抢了，为此90很是悔恨。
90喝着闷酒的时候很像是有故事的人，90为何小小年纪就独闯拉萨，我们没有去问，也没必要问，有些事不想说就不必说，有些痛不能提就不要提。
哪晚我已经忘了跟谁去了趟酒吧，喝了杯小酒，头晕脑胀，tv哥来找我们回去，他第二天要走了道个别，小凳子也走了。后来好多人都走了，离别是很伤感的一件事，港妹比我们晚一天走，而女神是最后一个。
哪天我去了新疆，继续我的长途旅程，老泥回了广州继续他的创作事业，90应该是回去继续他的皮条生意吧，而哪天tv哥在火车上遇到了他未来的媳妇。

又后来我又去了一趟拉萨，后来的后来我又去了一趟拉萨，都是路过。

关于在斯里兰卡被打劫哪件破事

发表于 2016-07-01   |   分类于 路上

关于在斯里兰卡被打劫那件破事

事情是这样的，哪天中午在康提湖边闲坐，被两个拿着写着各国语言的留言本的小黑哥搭讪，搭讪的目的就是带你装逼带你飞，带你看茶庄看景点。为了装逼为了飞，本着光顾国际友人顺便交个朋友，又是老司机带路的想法，那天鬼使神差就上车了。后来这种套路，在印度又遇到了一次。

其实当一直驶向山顶的时候，我已经觉得不对劲，脑袋里闪过无数个念头，猛虎组织？绑架勒索？鸡奸？跳车？爆头？弃尸荒野？每每当我想到死亡的时候反而觉得好受点，而我更害怕的是绑架勒索，到时候给国家添乱哪就麻烦了。

还好，我想，有点小庆幸，庆幸过关的时候没有兑换过多的美金。庆幸满嘴的胡子和穿着不太光鲜的衣裤，这使我看上去并不像是什么钱人，如果因为抢这么点钱而去杀一个人的话，我想他们应该不会这么笨。

他们会一直跟你说话，不知道是安慰他们自己的紧张情绪，还是安慰我。

动手是在山顶，把车驶向一个偏僻的角落。那时大家都很紧张，大声嚎着要钱包，我是怕他们会做出什么傻事，就给了过去。收走了所有的美金和3张毛爷爷，我害怕他们把我的手机收走或拿走我的银行卡逼我说出密码什么的，然而什么都没有发生，他们甚至没有拿走我的手机和相机。

整个过程过得很快，他们说会保证我安全并送我回去，只要我不报警，我也一直向他们保证我一定不会报警。
他们会制造各种令你感到不安的小动作，例如手里好像藏着点什么，发出点金属摩擦的声音，并且一直嚷嚷着I will kill you，I will kill you。中途想过跳车，但是我觉得我的小命应该算是保住了，也就放弃了跳车的念头，万一跳车不成功呢。

最后他们还是送我到了车站，下车的时候露出洁白的笑容，想想也算是很诚信。我活着回来了，打电话给talinga，告诉你们斯里兰卡人有多坏，眼泪也不自觉地掉了下来。

回到宿舍我还挣扎了很久要不要报警，给友人发信息好像也没什么卵用，最后还是打了大使馆电话，我还是不太诚信的。大使馆那边觉得人没事就好了。
我，应该不是第一个，也不是最后一个吧。

哪天的日落是红色的，整个康提都是红色的，有图，有照片，哪是见过美的傍晚。

白哈巴-蜜蜂

发表于 2016-05-19   |   分类于 路上

当我一脚把房门踢开的时候，发现房间多了一个人，正卷缩在被窝里咪着眼睛美滋滋地傻笑。一个一米多高的背包斜靠在床边，沾满泥土的徒步鞋散落一地。﻿

那是中午，一天中最好的阳光，透过纱窗洒落在地上，白炽的，温暖的，充满着活力与能量，微尘在空气中舞蹈着，像炊烟，徐徐升起，﻿

正直盛夏，然而炎热却与这里无关，身处祖国的北疆反而显得格外寒冷，刚搓洗过衣服的手还在发烫，精神上的伤痛也才刚刚缓解，在陌生人面前就更不好表现什么了。﻿

聊天搭讪是很自然的事，﻿

他说他来自乌鲁木齐，每年总会带妻子来喀纳斯两三次，一起走过禾木，喀纳斯，白哈巴。露营必需品都是他背，也难怪背这么大的背包。他的妻子就在旁边的房间，我问他为什么不住在一起？他说，贵，就一晚上也没必要去浪费。那天聊了很多，那时觉得他很幸福，像《东邪西毒》的洪七“谁说不能带着老婆走江湖”，策马扬鞭，人生几何。﻿

逃离城市的空气，清新得说不出话来，晚上十点的天空吐着鱼肚白，日落和日出也只不过是一个颠倒。﻿

那天的小木屋除了两个男人在说着梦话之外，还有一只小蜜蜂不知道什么时候闯了进来，然后又飞走了。
