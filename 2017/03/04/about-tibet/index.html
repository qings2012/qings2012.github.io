关于西藏

发表于 2017-03-04   |   分类于 路上

对于这个情怀已经聊烂的年代，我想聊一下情怀，关于西藏，拉萨，我们路过的青春、往事。

哪一年是2010年，暴动后的第二个春天，四月，春暖花开。我们是属于第二代进藏，这是按照大姐的说法，她和现在比较火的赵雷是属于第一代，去过西藏的人聊天之中都会不自觉地带点逼气。正如某人说，徒步的看不起骑车的，骑车的看不起搭车的，搭车的看不起坐车的，坐车的看不起坐火车的，坐火车的看不起坐飞机的，坐飞机的看不起所有人一样。
为了装逼，我和老泥手里端着两张开往拉萨的火车票什么都没准备就出发了。其实对于我们来说目标比较单纯，看一下布达拉，装个逼回去吹嘘一下也就够了。至于之后的什么羊湖什么纳木错大昭寺艳遇墙艳遇床我们是什么都不知道。
火车是56个小时，还记得同车是个女同志，去拉萨谋生，她说拉萨人很笨很懒，连土豆皮都不会剥，那边容易做生意哩！哪时的拉萨印象还只是停留在旧照片里面，破破旧旧想想应该就是哪样了。
火车经过唐古拉，高原阳光明媚，老泥的手机放着《坐上火车去拉萨》，哪时觉得相当的带劲，连隔壁箱的旅客也不知不觉啃上几句，坐上火车去拉萨，去看哪神奇的布达拉…

由于火车晚点，入住东措已经很晚，我们的房间是204。90是老房客，90之所以叫90因为他是90，哪个年代也就20不到。由于长得着急并且很老油条，刚开始我还以为90是专业拉皮条，开玩笑啦。还有一姐们叫老孙，后来我们还和老孙一起在长毛的酒吧门口喊过死了都要爱呐。港妹，港妹住在隔壁房间，爱好是睡觉，我们甚至组团去观摩。

藏地的阳光，艳遇墙下的艳遇，喇嘛的转经，八廓街上的讨价还价，累了就去光明茶馆抢地盘，吃着藏面，喝着甜茶，想着藏妹，吹着牛逼。分享淫生，分享艳遇，聊理想也聊淫生，失恋失意失业的三失青年，混在一起的日子并不孤单，萍水相逢的过客，纯纯的友谊。

每天都是聊天喝茶晒太阳，好不无聊，我们也由菜鸟成功转型为老鸟，指导菜鸟成为我们每晚必修课，如何装逼，如何装藏族，如何混进大昭寺，如何才能买到假佛珠假手镯等等都是我们的研究课题。90的皮(饭)条(局)业务也发展到隔壁，隔壁的隔壁，隔壁的隔壁的隔壁。港妹是被发展过来的。港妹来拉萨没有高反，一口气逛完布达拉大昭寺像吃生菜一样，这是她说的。哪天老孙约我们去见识见识尼泊尔餐顺便也约上了港妹，分摊消费，还不是因为穷吗。第一次见港妹，脚踏唐鞋，衣服是混搭，斜挎为人民服务背包，头戴牛仔帽，甚是有型，对于一个素食主义者来说还随身自带筷子，节约环保，我也是服了。后来老泥和港妹去了羊湖，我只好去纳木错，唉，好个重色轻友啊。
tv哥和小凳子也是90发展出来的。tv哥之所以叫tv哥是因为他衣服上面写着cctv，不是ccav。我们都是tv哥的忠实听众，tv哥是徒步拉萨，tv哥光芒万丈，tv哥开讲徒步经历万人空巷，我们都是90度仰望tv哥，tv哥是万人迷，tv哥把90的妹子全抢了，为此90很是悔恨。
90喝着闷酒的时候很像是有故事的人，90为何小小年纪就独闯拉萨，我们没有去问，也没必要问，有些事不想说就不必说，有些痛不能提就不要提。
哪晚我已经忘了跟谁去了趟酒吧，喝了杯小酒，头晕脑胀，tv哥来找我们回去，他第二天要走了道个别，小凳子也走了。后来好多人都走了，离别是很伤感的一件事，港妹比我们晚一天走，而女神是最后一个。
哪天我去了新疆，继续我的长途旅程，老泥回了广州继续他的创作事业，90应该是回去继续他的皮条生意吧，而哪天tv哥在火车上遇到了他未来的媳妇。

又后来我又去了一趟拉萨，后来的后来我又去了一趟拉萨，都是路过。
